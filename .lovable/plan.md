

# План: Fade-out hero-фото при скролле к отзывам

## Проблема

1. Spacer `h-[60vh]` создаёт **огромный зазор** между контентом и футером
2. Hero-фото всё равно видно при определённых условиях скролла
3. Это выглядит некорректно

## Решение

Вместо добавления spacer'ов — **анимировать opacity hero-фото**, чтобы оно плавно исчезало когда пользователь доскролливает до секции отзывов.

### Шаг 1: Удалить spacer

Убираем `h-[60vh]` spacer, который создаёт огромный зазор:

```tsx
// УДАЛИТЬ:
{isMobile && (
  <div className="bg-background h-[60vh] relative z-10" />
)}
```

### Шаг 2: Добавить fade-out анимацию для hero

Используем `reviewsInView` для управления opacity:

```tsx
<div
  ref={imageRef}
  className={`relative bg-secondary transition-opacity duration-500 ${
    isMobile 
      ? (isHeroSticky ? 'sticky top-9 z-0 h-[55vh]' : 'h-[55vh]') 
      : 'h-[75vh]'
  } ${isMobile && reviewsInView ? 'opacity-0' : 'opacity-100'}`}
>
```

Hero плавно исчезает за 500ms когда `reviewsInView` становится `true`.

### Шаг 3: Добавить bg-secondary fallback

Чтобы при исчезновении hero не было "дыры", добавим фоновый цвет, который сольётся с контентом:

```tsx
// Hero container уже имеет bg-secondary
// При opacity-0 фон остаётся, но картинка исчезает
```

Но лучше — сделать fade на **изображение**, а не на контейнер, чтобы фон оставался:

```tsx
<motion.img
  animate={{ 
    opacity: isMobile && reviewsInView ? 0 : 1 
  }}
  transition={{ duration: 0.5 }}
  // ...остальные props
/>
```

## Визуальный результат

```text
Скролл вверху:
┌────────────────┐
│    Hero 100%   │ ← полностью видно
├────────────────┤
│   Content      │
│   ...          │

Скролл до отзывов:
┌────────────────┐
│    Hero → 0%   │ ← плавно исчезает
├────────────────┤
│   Reviews      │
│   ...          │
├────────────────┤
│    Footer      │ ← нет зазора!
└────────────────┘
```

## Преимущества

- **Нет лишних spacer'ов** — контент плотно прилегает к футеру
- **Плавная анимация** — hero "растворяется" за 0.5 секунды
- **Чистый UX** — пользователь не видит hero из странного ракурса

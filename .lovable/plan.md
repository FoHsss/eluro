

# План: Header без фона — уходит целым модулем

## Проблема
При прокрутке текст Header уезжает, но белый фон (`bg-background/95`) остаётся видимым как полоса на фоне изображения товара. Это выглядит некрасиво.

## Решение
Убрать фоновый цвет с Header, сделать его полностью прозрачным. Тогда при прокрутке весь модуль будет визуально "исчезать" вместе с контентом.

---

## Изменения

### `src/components/Header.tsx`

**Было:**
```tsx
className="relative z-40 h-16 mt-9 bg-background/95 backdrop-blur-sm"
```

**Станет:**
```tsx
className="relative z-40 h-16 mt-9"
```

Убираем:
- `bg-background/95` — белый фон
- `backdrop-blur-sm` — размытие (не нужно без фона)

---

